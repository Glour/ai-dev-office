# Политика Codex CLI

Codex CLI — единственный утвержденный инструмент AI Dev Office для реализации и ревью кода.

## Агенты могут

- Создавать task briefs.
- Запускать wrappers из этой директории.
- Читать артефакты.
- Суммаризировать результаты.
- Возвращать неудачную работу Orchestrator.

## Агенты не должны

- Напрямую редактировать файлы целевых репозиториев.
- Запускать ad hoc code generation вне wrapper.
- Пропускать детерминированные проверки, если они есть в проекте.
- Скрывать failed tests, lint, build, migrations или review findings.

## Обязательный артефакт

Каждый запуск Codex CLI должен создать директорию с файлами:

- `task.md`;
- `run.json`;
- `stdout.log`;
- `stderr.log`;
- опционально `review.md`.
